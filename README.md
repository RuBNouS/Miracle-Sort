Miracle Sort com Interface em Python 

Vi um tiktok, achei bué funny, e decidi tentar fazer. Só que as coisas escalaram e agora tem mais funny (explicado a seguir).

O que é o Miracle Sort?
- Verifica se a lista de números está ordenada.
- Se não estiver, espera por um milagre (literalmente lol, espera que um raio cósmico atinja o computador e altere os bits da Ram na ordem exata para a lista ficar ordenada).
- Repete eternamente até o milagre acontecer.

Funcionalidades:
Meti mais funny em cima da piada original. Esta versão tem:
- Interface: Feita com `customtkinter`.
- Verificação Visual: Vê o algoritmo a testar a lista em tempo real. Os números ficam verdes se estiverem ordenados, se chegar a um número "errado" fica vermelho e recomeça, se a lista estiver ordenada ficam todos verdes e mostra mensagem de sucesso.
- Forçar Cosmic Ray: """"Simular"""" um raio cósmico para não ser preciso esperar milhões de anos.

A Roleta do "Cosmic Ray"

Quando clicas no botão Forçar Cosmic Ray, o universo muda os dados. Aqui estão as probabilidades do que pode acontecer:
|Probabilidade| O que acontece?|
|:---:|:---|
|70%| Radiação Real: Altera aleatoriamente (faz flip) a um bit de um dos números da lista.|
|10%| Documentação de Emergência: Abre o meu perfil [GitHub (RuBNouS)](https://github.com/RuBNouS) no browser.|
|10%| Rickroll: Abre o clássico Never Gonna Give You Up no YouTube.|
|5%| Erro Crítico: A aplicação explode.|
|5%| Erro Mega Crítico: Tenta desligar o teu computador (esta parte está comentada no código...).|

Como Executar:
1. [Python](https://www.python.org/downloads/) instalado.
2. A biblioteca necessária:
```bash
pip install customtkinter
```
3. Inicia o milagre correndo o script:
```bash
python "miracle sort.py"
```
